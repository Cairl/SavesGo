# SavesGo
“SavesGo”是款命令行程式，右Python语言编写。<br>
负责帮您管理受支持的软件/游戏，以快速备份/恢复其对应的配置/存档文件。

## 使用方式：
在控制台界面输入，如`save steam`或`load steam`，无大小写要求。<br>
受支持项目可从菜单列表中获知。
